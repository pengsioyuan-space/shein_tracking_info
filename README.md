# SHEIN 订单超时风险看板

## 本地运行

```bash
pip install -r requirements.txt
streamlit run dashboard/shein_order_risk_dashboard.py
```

## Streamlit Community Cloud 部署

- Repository: 你的 GitHub 仓库
- Branch: main
- Main file path: dashboard/shein_order_risk_dashboard.py

## 注意

不要上传真实订单 Excel、CSV 或 `store_key_all.txt` 密钥文件。
网页使用时直接上传 Excel 表格即可。
